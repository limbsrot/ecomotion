## Requirements
- Node.js 23 or later (from [Next.js docs](https://nextjs.org/docs/app/getting-started/installation#system-requirements))

  <br>

## Usage
1. **Install NodeJS:**
    - [Windows/macOS](https://nodejs.org/en/download/current)
    - [Linux](https://nodesource.com/products/distributions)

2. **Clone and Open the Project:**
    ```bash
    git clone https://github.com/limbsrot/ecomotion.git
    ```
    ```bash
    cd ecomotion
    ```
    ```bash
    npm install
    ```
6. **Start the Project:**
    ```bash
    npm run dev
    ```
