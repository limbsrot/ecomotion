## Requirements
- Node.js 23 or later (from [Next.js docs](https://nextjs.org/docs/app/getting-started/installation#system-requirements))
- Git (from [git docs](https://git-scm.com/install/))

  <br>

## Usage
1. **Install NodeJS:**
    - [Windows/macOS](https://nodejs.org/en/download/current)
    - [Linux](https://nodesource.com/products/distributions)
    
2. **Install Git**
    - [Windows/macOS/Linux](https://git-scm.com/install/))


4. **Clone and Open the Project:**
    ```bash
    git clone https://github.com/limbsrot/ecomotion.git
    ```
    ```bash
    cd ecomotion
    ```
    ```bash
    npm install
    ```
5. **Start the Project:**
    ```bash
    npm run dev
    ```
6. **Navigate to the page on applicable browser:**
    ```bash
    localhost:<port>
    ```
## How to run the project on your local network for testing on various devices.
```bash
npx next dev --hostname 0.0.0.0
```
